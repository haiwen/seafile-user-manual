    Introduction¶

    Seafile is an enterprise file hosting platform with high reliability and performance.

    Seafile contains a web interface for you to manage and share files. It has two kinds of desktop clients, the file syncing client and the drive client. It also has mobile clients for Android and iOS.

    You can download the clients in Seafile web site.

    If this manual does not contain the information you needed, you can seek help in our forum: https://forum.seafile.com .
